================================
 Turbo Shortcuts for Google拡張
================================

❏取り扱い種別

  フリーソフト



❏ソフトの概要

  あらかじめキーが固定されているChromeのショートカットキーを

  他のショートカットキーに割り当てたり、無効にすることができます。

  マウスクリックやホイールイベントにも対応しています。



  その他、タブ周りのコマンドや、ブックマークにショートカットキーを割り当てたり

  テキスト貼付けやCSS挿入、簡易JavaScript実行機能があります。



❏動作環境
  Windows版GoogleChrome

  OADG 日本語109/英語106キーボード




  WindowsXP32bit、Windows7-32bit、WIndows8-64bit環境で動作確認しています。

  Windows版以外のChromeや、Chrome以外の派生ブラウザでは動作致しません。



❏インストール方法
  Chromeのメニューからツール→拡張機能を開いて

  そのページへZIPファイルから解凍したCRXファイルをドラッグ&ドロップしてください。

  設定は拡張機能ページのオプションから行います。



❏アンインストール方法

  Chrome拡張機能ページから、ゴミ箱のアイコン（Chromeから削除）をクリックします。



❏注意事項
  - Alt+Tab、Ctrl+Esc、Win+EなどのWindowsショートカットキーは利用できません。

  - マウスホイールイベントでホイール高分解能マウスは未検証です。


  - PasteText、ExecuteScript、InsertCSS機能ではキーリピートは効きません。
